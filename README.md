# Wine-Clustering-Analysis
This project uses K-Means clustering to group Italian wines based on their chemical properties. The goal is to help the regional wine consortium understand natural groupings among three wine cultivars and support quality control and marketing decisions.

Dataset

wine_clustering.csv
Contains chemical measurements used to identify wine similarities.

Process

Load and clean data

Standardise features

Use Elbow or Silhouette method to select optimal clusters

Apply K-Means

Visualise clusters (e.g., PCA scatter plot)

Output

Cluster assignments for each wine

Visual cluster plot

Insights into wine similarities and differences
